## 验收 SOP

发版前对照本表逐项过一遍：

```bash
# 1. 编译
swift build -c release

# 2. 打包安装
bash build_app.sh

# 3. 启动 + 手动测试 L0～L4 每一项
open /Applications/快捷高光.app

# 4. 重点回归点（曾经踩过坑的功能必查）：
#    - F-004 上下方向 → 按住左 Option，看屏幕角落，方向应该和真实屏幕一致
#    - F-005 zoom 真值 → 拉到 4×、6×，看圈内是不是真的放大了
#    - F-204 锐化效果 → 圈中找到文字，关锐化和开锐化对比
#    - F-402 不弹权限 → 重 build 后启动应该不再弹屏幕录制授权框
```

任何一项 ❌ 不允许 push 到 main。
